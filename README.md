## How to run 
* First config your anaconda workspace, and open the anaconda prompt
  
* Second you need to install dependency  
  ```python
  pip install -r requirements.txt
  ```
* Third you need to install [vulkan sdk](https://vulkan.lunarg.com/) 

* Last you type 
  ```python
  ti cloth.py
  ```
